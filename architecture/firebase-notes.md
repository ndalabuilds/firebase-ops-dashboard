# Firebase Notes

## Suggested structure

- Firebase Authentication for login
- Firestore for records and dashboard data
- optional Cloud Functions for business logic
- optional hosting for deployment

## Example collections

- users
- leads
- tasks
- dashboard_metrics

## Example fields

- name
- status
- owner
- priority
- created_at
- updated_at

## Notes

This project is intended as a lightweight internal tool pattern for small business operations.
